# RealtimeDetectionMVP

This is the isolated iOS app copy used by the `final/` Leonardo USB MIDI prototype.

Use this folder when you want to:

- open the final prototype in Xcode
- edit the player-selection / deadzone / USB MIDI app
- bundle the exported `detector.pte` into the app target

- `RealtimeDetectionMVP.xcodeproj`
